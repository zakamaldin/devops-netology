    ### Ответ:

    ```
    modprobe -v dummy numdummies=2
    ip addr add 172.17.0.150/24 dev dummy0
    ip link add dummy0 type dummy
    ip link set dummy0 up
    ip route add 172.17.0.150 dev dummy0
    ```
